Obtain a list of invoices by quarter and year for customers who exceed a certain amount.

== Description ==

Obtain quarterly summary, annual summary and list of invoices per client that exceed a user-selectable amount.
It is necessary to have installed the WooCommerce PDF Invoices & Packing Slips plugin.
You can choose up to 5 years ago and the amount you want to generate the listings.

== Installation ==

1. Upload the "Invoices by Customer" plugin to the / wp-content / plugins / or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Access the plugin through the "Menu->Woocommerce->Invoices by Customer" to generate the listings


== Screenshots ==

1. Select the minimum annual billing amount per customer and choose the year.
2. List invoiced by customer, quarter and year. Also detailed listing of all invoices per customer.
